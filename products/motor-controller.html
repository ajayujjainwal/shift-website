Overview

The SHIFT Motor Controller Platform is an inhouse-developed, scalable embedded control platform for DC brush and BLDC motors used across body actuation applications ‚Äî electric seats, power windows, tailgate, HVAC blend flaps, and motorized side steppers. The platform implements a common hardware base with configurable firmware, enabling Lumax to address multiple actuation requirements on a single validated design and reduce OEM integration effort.

Key Features
  • Scalable platform ‚Äî supports DC brush, BLDC, and stepper motor topologies
  • Configurable current, torque, and position control loops per application
  • CAN 2.0B / LIN bus integration for body domain ECU commands
  • Integrated gate driver and FET output stage ‚Äî compact single-board design
  • Thermal and current protection with adjustable thresholds
  • Anti-pinch protection for windows and power tailgate
  • 12V/24V supply support ‚Äî suits passenger and light commercial vehicles
